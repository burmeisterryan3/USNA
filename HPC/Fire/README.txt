README for fire.c

COMPILE
  To compile, utilize "make".
  An executable named "fire" will be created.

RUN
  To run, utilize "make run" after compiling.
  The process will run a 2000x2000 point simulation across 16 processors.

CLEAN
  To clean-up the local directory, utilize "make clean".
  This will remove temporary files as well as the executable.
